# StableStreams

Real-time cryptocurrency streaming payments for automated payroll, freelancer compensation, and continuous token distributions.

## What is StableStreams?

StableStreams enables continuous, on-chain token transfers - money streaming directly to wallets in real-time. Instead of monthly lump-sum payments, funds flow continuously every second.

## Why Streaming Payments?

### For Employers & DAOs
- **Automated Payroll**: Set it and forget it. No more manual monthly transfers
- **Reduced Overhead**: Smart contracts handle everything autonomously
- **Global Access**: Pay anyone, anywhere, instantly - no banking infrastructure needed
- **Transparency**: All transactions are on-chain and auditable

### For Recipients
- **Immediate Access**: Get funds as they're earned, not at the end of the month
- **Financial Flexibility**: No waiting periods or payment delays
- **Full Transparency**: See exactly what's being streamed in real-time

## Use Cases

- **Crypto Payroll**: Pay employees and contractors in USDC, DAI, or ETH with continuous streams
- **Vesting Schedules**: Grant tokens that unlock gradually over time
- **Freelance Payments**: Pay for ongoing work as it happens
- **Royalty Distributions**: Automatically distribute earnings to multiple recipients

## Features

- Real-time streaming to any wallet address
- Multi-chain support (Polygon, Polygon Amoy)
- Stablecoin support (USDC, DAI)
- Native ETH streaming
- Stream management (create, monitor, cancel)
- Live rate aggregation dashboard

## Getting Started

```bash
npm install
npm run dev
```

Connect your wallet, set your stream parameters (recipient, amount, duration), and start streaming immediately.

## Technology

- **Frontend**: React, Vite
- **Web3**: Wagmi, Viem
- **Smart Contracts**: Solidity (Polygon)
- **Styling**: Custom CSS with noise textures

## Network

- **Mainnet**: Polygon
